# CineBot

Telegram-бот для поиска, отслеживания и получения рекомендаций по фильмам. Включает встроенное мини-приложение (Telegram Web App).

---

## Запуск

```bash
NO_PROXY="*" python movies_bot.py
```

`NO_PROXY="*"` обязателен — на машине настроен системный прокси, который блокирует исходящие соединения.

При старте бот автоматически поднимает Cloudflare-туннель. В консоли появится строка:

```
Mini-app: https://xxxx.trycloudflare.com
```

После этого отправь `/start` в Telegram, чтобы получить актуальную клавиатуру с кнопкой мини-приложения.

---

## Команды и функции

| Кнопка | Описание |
|--------|----------|
| Поиск | Подбор фильмов по жанру, длительности и возрастному рейтингу |
| По названию | Поиск конкретного фильма |
| Топ | Популярные фильмы за неделю или месяц, отфильтрованные по любимым жанрам пользователя |
| Подборка | ИИ-рекомендации на основе 5 последних просмотренных |
| Похожие | ИИ подбирает 5 фильмов, похожих на указанный |
| Просмотренные | Список просмотренных с датой и оценкой, постраничная навигация |
| Профиль | Количество просмотров и топ-5 жанров пользователя |

---

## Мини-приложение

Открывается кнопкой внутри Telegram через Cloudflare-туннель.

| Вкладка | Описание |
|---------|----------|
| В тренде | Топ-20 фильмов недели по данным TMDB |
| Поиск | Полнотекстовый поиск по названию |
| Просмотренные | История с оценками |
| Хочу посмотреть | Вишлист |

В карточке фильма доступны действия: отметить просмотренным, поставить оценку (1–10), добавить в вишлист.

## Стек

| Компонент | Технология |
|-----------|------------|
| Бот | python-telegram-bot |
| HTTP | httpx |
| База данных | SQLite (stdlib) |
| Источник данных | TMDB API |
| ИИ-рекомендации | DeepSeek via OpenRouter |
| Туннель | cloudflared (trycloudflare.com) |

---

## База данных

Файл `movies_bot.db` создаётся автоматически в директории скрипта.

| Таблица | Содержимое |
|---------|------------|
| `users` | Telegram ID, статистика жанров (JSON) |
| `watched` | Просмотренные фильмы, дата, оценка |
| `wishlist` | Фильмы в списке желаний |

---

## Известные ограничения

- **URL туннеля меняется при каждом перезапуске.** После рестарта бота необходимо повторно отправить `/start`, чтобы получить кнопку с актуальным URL.
- **Одновременно должен работать только один экземпляр бота.** При запуске второго возникает ошибка `409 Conflict` от Telegram API.
- **DNS-ошибки в `cloudflared_log.txt`** (`lookup region1.v2.argotunnel.com: i/o timeout`) некритичны — туннель продолжает работать.
